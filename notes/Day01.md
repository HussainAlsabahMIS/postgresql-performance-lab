\# Day 01 — Getting started with PostgreSQL



\## What I did

\- Installed PostgreSQL and pgAdmin

\- Connected successfully to the local server

\- Created my first database: `booktrade\_pg`

\- Opened the Query Tool



\## What I learned

\- PostgreSQL uses `SERIAL` instead of MySQL's `AUTO\_INCREMENT` for

&#x20; auto-incrementing primary keys

\- Basic `CREATE TABLE`, `INSERT`, and `SELECT` syntax

\- First impression: PostgreSQL syntax is close to standard SQL,

&#x20; but stricter than MySQL in some ways
